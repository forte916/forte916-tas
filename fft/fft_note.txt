* SCPH1000.BIN では 9f  待つと 199/199
* SCPH1001.BIN では ??


game start
----------

* 706f : start , skip title
* 742f : start , new game
* 935f : start , skip message

  * after 324f, determined ramza status
  * after 25f, show input view.

* input name
  * skip 3f
  * press [], delete 1st letter
  * skip 10f

  * skip 3f
  * press [], delete 2nd letter
  * skip 10f

  * skip 3f
  * press start
  * skip xx
